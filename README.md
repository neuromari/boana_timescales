# About the repository: boana_timescales

This is a repository containing code and data for our paper:

Rodriguez-Santiago, M., Pouso, P., Hoke, K., & Zornik, E. (2026). Timescales of call variability in a South American treefrog. *Journal of Comparative Physiology A*, *in press*

## How to cite

If you use these data or code, please cite the paper and the archived repository as follows:

**paper**

Rodriguez-Santiago, M., Pouso, P., Hoke, K., & Zornik, E. (2026). Timescales of call variability in a South American treefrog. *Journal of Comparative Physiology A*, *in press*

**code**

Rodriguez-Santiago, M. (2026). **boana_timescales: Code and data for *Timescales of call variability in a South American treefrog*** (Version 1.0.0) [Data set]. Zenodo. <https://doi.org/10.5281/zenodo.21269687>

Repository: <https://github.com/neuromari/boana_timescales>

## Contents

- ./analysis/ - contains custom scripts for analysis, analysis scripts are run by timescales separately, and code for figures requires all '01\_' scripts to be fully run first

- ./data/ - contains raw automated selection .txt files and final combined .csv by timescale

- ./manuscript/ - contains final manuscript figures merged and finalized in Adobe Illustrator
